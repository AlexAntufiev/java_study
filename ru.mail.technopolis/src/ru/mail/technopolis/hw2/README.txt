Домашнее задание #2
ДЗ состоит из нескольких пунктов:
Разобрать входное арифметическое выражение, заданное в виде строки, на лексемы и представить его в виде примитивов, подобных тем, что мы разбирали на практике: Const, Variable, Add, Multiple, ...
По вызову метода evaluate с некоторым значением переменной "x", выдавать результат этого выражения
Вы должны уметь работать с любым арифметическим выражением, поданным на вход (ограничения заданы ниже)
Приблизительный код домашнего заданий можно найти на дропбоксе (ссылка выше)

Детали реализации:
В записи выражения могут встречаться:
умножение (*)
деление (/)
сложение/вычитание (/)
унарный минус (-)
константы
круглые скобки
переменная х
любое количество пробельных символов в любом месте
Приоритет операций: унарный минус, умножение/деление, сложение/вычитание
Если выражение составленно не корректно, вам необходимо вывести строку "Некорректная запись"
Предполагается, что разбор выражения будет происходить с использованием метода рекурсивного спуска, работающего за линейное время
, но, разумеется, можно использоваться и свой алгоритм,
Например, результаты выполнения вашей программы для числа 5 на следующих выражениях:
"5 * ( x - 2 ) + 3" = 18
"5 ** (x)" = Некорректная запись
"-3 * x + 4" = -11
