Домашнее задание #3

Необходимо разработать класс RecursiveGrep, осуществляющий поиск строки в файлах (по аналогии тому, что делали на практике)

Требования:
Формат запуска программы :
1
java RecursiveGrep <входной файл> <выходной файл> <искомая строка>
 
Входной файл содержит список файлов или директорий в которых необходимо искать входную строку
Выходной файл содержит результат для каждого найденного файла в формате :
1
<номер строки на котором нашлась искомая строка> <путь до файла>
 
Не все файлы можно целиком загрузить в операционную память - это означает то, что имеет смысл считывать данные из файла не все сразу, а по частям

Пример:
Входной файл:
1
2
3
4
/home/ilya.dronov/polis/homework6/
/home/ilya.dronov/polis/homework6/test.txt
/home/ilya.dronov/polis/homework6/samples
/home/ilya.dronov/polis/homework6/samples/1.txt
 
Выходной файл:
1
2
3
line #3 /home/ilya.dronov/polis/homework6/sample.txt
line #292 /home/ilya.dronov/polis/homework6/test.txt
line #400 /home/ilya.dronov/polis/homework6/samples/2.txt
 


Критерии:
Корректная работа программы
Правильная обработка исключений
Отсутствие утечек памяти
Для сдачи необходимо иметь тестовые данные по которым ведется поиск: не менее 3 директорий, не менее 5 файлов
